Terri's Example: Applying CSS to HTML With Cascade and Specificity

My cat Tina is a pain when it comes to wanting her breakfast first thing in the morning!
Finny, our other cat, will wait patiently without meowing and scratching at the door.

What is demonstrated on this page:

  • the bold text is red because the "strong" style in the styles.css has a red color and I gave it an !important flag. That over-rides everything!
  • If you comment out the strong style on line 12 of styles.css, then you get green text for the "Finny" text because it takes the "highlight" class color of green.
  • "Pain" would then be orange because it has an inline style of orange for the strong tag.
  • "Scratching" will just be blue because it has no class and would take the last color assigned to div tags in the styles.css file...which is blue!
